# supra-landing

Landing page for a restaurant booking platform, focused on clear conversion and real-world usability.

---

## Overview

**supra-landing** is a conversion-focused landing page designed to introduce and validate a restaurant reservation system.

It communicates the product’s value to both restaurants and customers, with emphasis on simplicity, control, and real operational benefit.

---

## Features

- Clear product positioning
- Dual audience targeting (restaurants / customers)
- Conversion-driven layout
- Mobile-first responsive design
- Fast-loading, minimal UI

---

## Purpose

The goal of this landing page is to:

- Explain the booking system quickly
- Demonstrate value over manual reservations
- Drive onboarding for restaurants
- Serve as the public entry point to the platform

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## Sections

- Hero (core value proposition)
- How it works
- Benefits for restaurants
- Benefits for customers
- Social proof / trust layer
- Call to action

---

## Development Approach

Built with a focus on:

- clarity over complexity  
- fast comprehension  
- strong visual hierarchy  
- minimal friction to conversion  

The implementation follows a component-based structure for easy iteration and scaling.
